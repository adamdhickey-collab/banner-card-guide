Developer Implementation Notes — Sitecore SXA

How the flexbox pattern keeps CTAs aligned across cards with varying content lengths — without fixed heights.

The Core Flexbox Pattern

Apply this to both card types. The grid's default align-items: stretch makes every card in a row grow to the same height. Flexbox then distributes that height internally so the CTA always pins to the bottom regardless of how much text is above it.

  • Card wrapper: display: flex; flex-direction: column;
  • Description element: flex: 1; — expands to fill all available vertical space
  • Actions wrapper: margin-top: auto; — pushes the CTA to the card bottom
⚠ Do not set height, min-height, or max-height on the card. On mobile, cards stack into a single column and content becomes much shorter — a fixed height creates unnecessary dead space below the CTA.
